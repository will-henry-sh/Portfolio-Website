# Twenty Twenty-Five WHC Child Theme

## Activate
1. Go to `Appearance -> Themes` in WP Admin.
2. Activate `Twenty Twenty-Five WHC`.

## Live Update Workflow
1. Edit files in `wp-content/themes/twentytwentyfive-whc/`.
2. Deploy/sync that child theme folder to live.
3. Keep parent theme `twentytwentyfive` updated from WordPress normally.

## Overridden Files
- `style.css`
- `functions.php`
- `templates/front-page.html`
- `parts/site-footer.html`
